EM Designs — موقع جاهز للنشر (Black & Gold Premium)

كيفية النشر على GitHub Pages:
1) ادخلي على الريبو في GitHub.
2) اضغطي Add file → Upload files ثم ارفعي كل الملفات (index.html و images/*).
3) اضغطي Commit changes (الالتزام بالتغيّرات).
4) افتحي Settings → Pages واختر Main branch ثم Save.
5) بعد قليل سيظهر رابط الموقع: https://<username>.github.io/<repo>/

إذا احتجتي أعدل بيانات التواصل أو أحط صورك الحقيقية، قوليلي وانا أعدلها لك مباشرة.
