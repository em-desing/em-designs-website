تعليمات النشر على GitHub Pages:
1) افتح الريبو في GitHub.
2) اضغط Add file → Upload files، ثم ارفع جميع الملفات الموجودة هنا (index.html + style.css + images/*).
3) اضغط Commit changes.
4) اذهب إلى Settings → Pages، اختر Main branch → Save.
5) افتح الرابط الذي سيظهر لمعاينة الموقع.
