# EM Design Website

موقع احترافي لعرض أعمال تصميم الجرافيك والديكور.

## صفحات الموقع
- **الرئيسية (index.html)**: صفحة الترحيب وعرض لمحة عن خدمات EM Design.
- **عننا (about.html)**: صفحة تقديم نبذة عن الفريق والخدمات.
- **أعمالنا (portfolio.html)**: صفحة معرض المشاريع والأعمال السابقة.
- **تواصل معنا (contact.html)**: صفحة تحتوي على معلومات التواصل.

## تصميم الموقع
- ملف CSS رئيسي: `style.css`  
- كل الصور موجودة داخل مجلد: `images/`  
- التصميم بروفيشنال مع ألوان وخطوط متناسقة.

## كيفية استخدام المشروع
1. افتح أي صفحة HTML في متصفحك لمعاينة الموقع.
2. لتغيير الصور، استبدل الصور الموجودة في مجلد `images/` بصورك الخاصة بنفس الأسماء.
3. لتعديل التصميم، افتح ملف `style.css` وعدل عليه حسب الحاجة.

## روابط GitHub Pages
بعد رفع المشروع على GitHub، يمكنك تفعيل GitHub Pages للحصول على الموقع أونلاين.
